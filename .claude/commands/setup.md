Set up this dotfiles repo on the current machine.

1. Read install.sh to understand what it does
2. Run `./install.sh` and handle any errors
3. Verify symlinks are correct with `ls -la ~/.zshrc ~/.config/starship.toml ~/.config/yazi/theme.toml ~/.config/zellij/config.kdl ~/.gitconfig`
4. Run `source ~/.zshrc` to activate
5. Report any issues
