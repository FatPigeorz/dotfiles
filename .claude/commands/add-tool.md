Add a new tool's config to the dotfiles repo.

1. Identify the tool's config file location on this machine
2. Read the current config
3. Create a directory under this repo: `<tool>/`
4. Copy the config into it
5. Determine which functional category it belongs to (toolchains, terminal, files, editor, git, theme)
6. Update CLAUDE.md: add the tool under the correct category with install + config instructions
7. If the tool supports rose-pine, configure it and document in `theme/README.md`
8. Update README.md

Tool name: $ARGUMENTS
